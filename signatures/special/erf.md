## Scipy

```
scipy.special.erf(z, out=None) → ndarray
```

Supports complex.

## CuPy

```
cupy.scipy.special.erf(z, out=None) → ndarray
```

## JAX

```
jax.scipy.special.erf(x) → ndarray
```

## PyTorch

```
torch.special.erf(input, *, out=None) → Tensor
```

Also aliased as `torch.erf`.
