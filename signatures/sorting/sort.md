# sort

## NumPy

```
numpy.sort(a, axis=-1, kind=None, order=None) → ndarray
```

## CuPy

```
cupy.sort(a, axis=-1) → ndarray
```

```

```


## PyTorch

```
torch.sort(input, dim=-1, descending=False, out=None) → (Tensor, LongTensor)
```

## Tensorflow

```
tf.sort(values, axis=-1, direction='ASCENDING', name=None) → Tensor
```
