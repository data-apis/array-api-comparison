# tril

## NumPy

```
numpy.tril(m, k=0) → ndarray
```

## CuPy

```
cupy.tril(m, k=0) → ndarray
```

## dask.array

```
dask.array.tril(m, k=0) → ndarray
```

## JAX

```
jax.numpy.tril(m, k=0) → ndarray
```

## MXNet

```
np.tril(m, k=0) → ndarray
```

## PyTorch

```
torch.tril(input, diagonal=0, *, out=None) → Tensor
```

Supports batching.
